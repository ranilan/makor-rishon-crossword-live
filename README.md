# תשבץ מקור ראשון

אפליקציית פתרון קטנה לתשבץ היגיון מקור ראשון של דקל בנו.

גרסה ראשונה:

- תשבץ 11x11.
- מילוי תאים במובייל וב-PC.
- שמירת התקדמות מקומית בדפדפן.
- מעבר בין הגדרות מאוזן/מאונך.
- המרה אוטומטית של אותיות סופיות לאותיות רגילות.
- בדיקת מילה או כל התשבץ כאשר פתרונות זמינים.
- ארכיון ושיתוף קישור.
- קישור לתמונה ול-PDF המקוריים.
